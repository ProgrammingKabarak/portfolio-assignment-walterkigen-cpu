## Project Description
A responsive, single-page personal portfolio website showcasing my background,
technical skills, projects, resume, and contact information. Built as the
capstone assessment for Web Programming I.

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, custom properties, media queries)
- Vanilla JavaScript (no frameworks/libraries)

## Features Implemented
- Responsive navigation menu (mobile hamburger toggle)
- Typing animation on the hero name
- Dark / light theme toggle (persisted via localStorage)
- Scroll-reveal animations for each section
- Animated skill bars
- Scroll-to-top button
- Contact form with real-time JavaScript validation
- Fully responsive layout: mobile, tablet, laptop, desktop
- Accessible markup: semantic HTML, alt text, form labels, visible focus states,
  sufficient color contrast, keyboard navigation
